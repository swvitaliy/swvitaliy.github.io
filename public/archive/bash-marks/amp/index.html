Переход по директориям в bash

swvitaliy - 14 дек. 2021

Нечеткий поиск

Следующий алиас дает возможность быстро перейти на нужную директорию, вводя только часть символов пути деректории, а так же посмотреть содержимое этой директории

cd_fzf() {
  d=$(fd -t d | fzf --preview 'ls -lh {}' -q "${1}")
  [[ "${d}" != "" ]] && cd "${d}"
}
alias cf='cd_fzf'

В данном алиасе используются 2 программы:

  • fzf – утилита для нечеткого поиска
  • fd – аналог find с человекопонятным синтаксисом

Использование меток

Чтобы быстро переходить от одной директории/проекта к другой, находящейся с точки зрения пути неблизко, я использую следующий bash скрипт, который добавлен в ~/.bashrc. Суть его в том, что в директории ~/.marks можно быстро создавать/удалять/просмотреть симлинки и переходить по ним. Использую уже около 3-х лет или около того и могу сказать, что крайне удобная штука.

Изначальный скрипт взял откуда-то из сети (не нашел откуда) и доработал функцию marks (добавил выбор метки с переходом). В интернетах есть ещё несколько вариантов того же самого, но прижился как-то именно этот вариант.

export MARKPATH=$HOME/.marks

export LC_COLLATE=ru_RU.UTF-8
export LC_CTYPE=ru_RU.UTF-8

function jump {
  if [[ "${1}" == "" ]]; then
    marks
  else
    cd -P "$MARKPATH/$1" 2>/dev/null || echo "No such mark: $1"
  fi
}

function mark {
  mkdir -p "$MARKPATH"; ln -s "$(pwd)" "$MARKPATH/$1"
}

function unmark {
  rm -i "$MARKPATH/$1"
}

function marks {
  if [[ "${1}" != "" ]]; then
    mark "${1}"
  else
    PS3="goto> "
    MARKLIST=$(ls -ltr "$MARKPATH" | sed 's/ / /g' | awk -F' ' '{if (NR > 1) print $9" -> "$11;}')
    options=()
    while read -r line
    do
      options+=("$line")
    done < <(printf '%s\n' "$MARKLIST")
    COLUMNS=12
    select ma in "${options[@]}";
    do
      mm=$(echo "${ma}" | cut -d' ' -f1 -)
      jump "${mm}"
      break;
    done
  fi
}

alias j='jump'
alias um='unmark'
alias m='marks'

_completemarks() {
   local curw=${COMP_WORDS[COMP_CWORD]}
   local wordlist=$(find $MARKPATH -type l -printf "%f\n")
   COMPREPLY=($(compgen -W '${wordlist[@]}' -- "$curw"))
   return 0
}

complete -F _completemarks jump unmark

Чтобы создать симлинк текущей директории необходимо выполнить команду mark название_метки или m название_метки.

Чтобы удалить метку нужно выполнить команду unmark название_метки или um название_метки.

Чтобы перейти по метке нужно выполнить команду jump название_метки или j название_метки.

Чтобы просмотреть список меток и перейти по одной из них можно использовать команду marks или просто m или j. Вам будут предложены варианты перехода с названиями меток, путями и цифрой, которую нужно ввести для перехода.

В планах есть переключаться между воркспейсами, поскольку в dotnet принято много проектов создавать и они захламляют пространство имен. Предположительно, для этого буду использовать поддиректории ~/.marks/{workspace} и хранить имя текущего в файле ~/.marks/current_workspace.
