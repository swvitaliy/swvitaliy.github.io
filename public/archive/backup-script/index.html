Бекап директории с помощью rsync

2 июля 2022 г.

Ниже представлен простой скрипт бекапа директории с помощью rsync, sshfs, а так же tailscale в качестве vpn. Параметры запуска чрезвычайно просты

  1. директория, которую необходимо сохранить
  2. куда выполнять бекап
  3. удаленная директория, на которую выполнять бекап

Данный скрипт сначала выполняет соединение с vpn, если оно не было установлено, затем монтирует удаленную директорию через sshfs, затем выполняет rsync. Так же он показывает нотификации в gnome, посредством notify-send, статус (успешно, ошибка),  длительность выполнения.

Пример вызова в cron:

30 0 * * * sudo -u vit /home/vit/backup.sh >> /home/vit/backup.log /home/vit/Projects /home/vit/backups rockpi:///home/rock/backups

В случае, если вам нужно добавить директорию, можно создать файл backup.list и пройтись по нему циклом:

function rsync1 () {
  rsync -av --exclude="node_modules" --exclude="bower_components" --no-links "${1}" "${2}"
}

if [[ -f "${src}" && "${src}" == *".list" ]]; then

    rsync1 "${i}" "${dst}"
  done
else
  rsync1 "${src}" "${dst}"
fi
